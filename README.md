# Dasho

Simple and customizable Dashboard, made with Ionic, GraphQL and angular5 (former CAS_FEE@HSR Diploma Project 2016).

## Getting Started

### Prerequisites

- Node >= 10.0

### Installing

```
npm install -g ionic mocha
```

```
npm install
```

## How to start

Frontend

```
ionic serve
```

Backend

```
node server
```

## Running the tests

```
npm test
```
